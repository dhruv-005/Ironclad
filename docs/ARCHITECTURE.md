# 🏗️ Ironclad System Architecture

## Overview
Three-layer architecture: Perception, Intelligence, Brain

### Perception Layer
- InsightFace RetinaFace detection
- ArcFace 512-D embeddings
- FAISS HNSW vector search

### Intelligence Layer
- DuckDuckGo OSINT search
- 15+ targeted queries
- Social media detection

### Brain Layer
- LangChain ReAct Agent
- Ollama local LLM
- FastMCP server bridge
