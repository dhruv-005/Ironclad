## 🚀 Quick Start in 60 Seconds

> **Prerequisites:** Python 3.10+, Git, pip

|:----:|--------|---------|
| **1** | Clone the repository | `git clone https://github.com/dhruv-005/Ironclad.git && cd Ironclad` |
| **2** | Install dependencies | `pip install -r requirements.txt` |
| **3** | Initialize the database | `python init_db.py` |
| **4** | Launch the application | `streamlit run app.py` |
| **5** | Open in your browser | `http://localhost:8501` |

```bash

    libgl1 libglib2.0-0 libsm6 libxext6 \

    build-essential git curl

# 2. Clone the repository
```

## 🐛 Troubleshooting

<details>
<summary><b>ModuleNotFoundError: No module named 'faiss'</b></summary>

<br/>

```bash
pip install faiss-cpu --no-cache-dir

# If that fails on Linux, install OpenBLAS first
sudo apt-get install libopenblas-dev
pip install faiss-cpu --no-cache-dir
```

</details>

<details>
<summary><b>ImportError: libGL.so.1: cannot open shared object file</b></summary>

<br/>

```bash
sudo apt-get install libgl1 libglib2.0-0 libsm6 libxext6
pip uninstall opencv-python -y
pip install opencv-python-headless --no-cache-dir
```

</details>

<details>
<summary><b>OllamaConnectionError: Cannot connect to Ollama server</b></summary>

<br/>

```bash
# Start the Ollama server in a separate terminal
ollama serve

# Pull a model if you haven't already
ollama pull tinyllama

# Alternatively, switch to Direct Mode in the sidebar — no Ollama needed
```

</details>

<details>
<summary><b>FaceDetectionError: No face detected in image</b></summary>

<br/>

```
· Use a clear, frontal face photograph
· Minimum image resolution: 100 × 100 pixels
· Ensure adequate lighting with no harsh shadows
· Face must be fully visible in the frame
· Avoid extreme angles (> 45° tilt)
· Supported formats: JPG · PNG · JPEG only
```

</details>

<details>
<summary><b>GitPushError: Updates were rejected by remote</b></summary>

<br/>

```bash
git pull origin main --allow-unrelated-histories --no-edit
git push -u origin main
```

</details>

<details>
<summary><b>ImportError: pickle5 version incompatible</b></summary>

<br/>

```bash
sed -i '/pickle5/d' requirements.txt
pip install -r requirements.txt
```

</details>

<details>
<summary><b>StreamlitError: Port 8501 already in use</b></summary>

<br/>

```bash
# Free the port
sudo fuser -k 8501/tcp

# Or launch on an alternate port
streamlit run app.py --server.port 8502
```

</details>
