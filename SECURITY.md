# 🔒 Security Policy - Ironclad

## Supported Versions
| Version | Supported |
|---------|-----------|

## Reporting Vulnerabilities
Open a GitHub Issue with label: security

## Privacy Notice
- Ironclad runs 100% locally
- No data sent to external servers
- All embeddings stored locally only
- No cloud dependencies

## Ethical Use Policy
- For authorized investigations only
- Do not use without proper consent
- Comply with local privacy laws
